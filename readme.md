untuk memulai server: <br>
-> buat folder dengan nama 'server'<br>
-> inisialisasi package.json dengan menjalankan perintah 'npm init -y' di terminal<br>
-> install depedensi yang diperlukan :<br>
  -> 'npm i express sequelize mysql2 bcryptjs jsonwebtoken cors dotenv'<br>
  -> 'npm i -g nodemon sequelize-cli'<br>
-> setup sequelize 'sequelize init'<br>

<br>
untuk memulai client:<br>
-> install react dengan perintah 'npm create vite@latest'<br>
  -> masukkan nama 'client'<br>
  -> react<br>
  -> javascript<br>
