untuk memulai server:
-> buat folder dengan nama 'server'
-> inisialisasi package.json dengan menjalankan perintah 'npm init -y' di terminal
-> install depedensi yang diperlukan :
  -> 'npm i express sequelize mysql2 bcryptjs jsonwebtoken cors dotenv'
  -> 'npm i -g nodemon sequelize-cli'
-> setup sequelize 'sequelize init'


untuk memulai client:
-> install react dengan perintah 'npm create vite@latest'
  -> masukkan nama 'client'
  -> react
  -> javascript
